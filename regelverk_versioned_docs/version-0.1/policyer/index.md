# Policyer

Policyerna operationaliserar villkorsbilagornas krav till harmoniserade processer. De säkerställer att exempelvis anslutning och registrering genomförs på ett enhetligt sätt inom alla federationskontexter.

Följande policyer ingår:

- [**Anslutningspolicy för federationsmedlemmar**](/regelverk/policyer/anslutningspolicy-federationsmedlemmar) — krav och kontroller vid verifiering av organisationsidentitet och anslutning av federationsmedlem.
- [**Anslutningspolicy för anslutningsoperatörer**](/regelverk/policyer/anslutningspolicy-anslutningsoperatorer) — krav och kontroller vid godkännande av anslutningsoperatör *(skelett)*.
- [**Registreringspolicy**](/regelverk/policyer/registreringspolicy) — hur tekniska komponenters metadata registreras, ändras och avregistreras *(skelett)*.
