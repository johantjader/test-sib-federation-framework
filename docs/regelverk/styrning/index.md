# Styrning

:::info Uppdaterad i Regelverk 0.2
Detta dokument har uppdaterats. Välj version i versionsväljaren ovan för att jämföra med tidigare versioner.
:::

Styrningsdokumenten anger formella kriterier för strategiska vägval inom federationsplattformen, till exempel vid etablering av nya federationskontexter.

Följande styrningsdokument ingår:

- [**Etablering av federationskontexter**](/regelverk/styrning/etablering-federationskontexter) — principer och kriterier för när och hur nya kontexter får etableras.
